Simple python script that extracts the data from the given csv file, and uses
numpy library to calculate descriptive statistics, mean, median, and mode. It then
separates each variable into its own array, so that distribution can be visualized 
on a histogram.
----------------
How to run:
Navigate to the directory, you may need to:
pip install matplotlib
pip install numpy
pip install scipy

then:
python3 stats.py
